# Курс по Claude Code для бизнес-задач

Практический курс по применению Claude Code в задачах, не связанных с разработкой.

## Уроки

- [Урок 1](demo/lesson1/) — введение, тренды AI, сложность задач, промпты
- [Урок 2](demo/lesson2/) — команды, работа с документацией, пользовательские агенты
- [Урок 3](demo/lesson2/) — продолжение практического занятия урока 2 «Работа с документацией», базовые промпты
- [Урок 4](demo/lesson4/) — дизайнер и код: стилизация компонентов через ИИ
- [Урок 5](demo/lesson5/) — практика: стилизация компонентов через ИИ

Новые уроки добавляются по мере проведения курса.
