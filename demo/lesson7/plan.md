# Урок 7 -- Практика: GitLab через OpenCode

## Формат

Один час: ~20 мин теория и настройка + ~30 мин практика + ~10 мин кейс.

## Часть 1. Подключение OpenCode к GitLab через glab (~10 мин)

- Что такое glab: CLI для GitLab, аналог gh для GitHub -- агент использует его для работы с issues, merge requests, репозиториями
- Установка glab
- Авторизация через glab auth login
- Проверка подключения: попросить агента показать список проектов

## Часть 2. Базовые действия с GitLab через OpenCode (~10 мин)

Демонстрация на живом примере:
- Пул из GitLab
- Создание новой ветки
- Поиск по проекту
- Анализ дифов
- Коммит и пуш в remote

## Часть 3. Практические задачи (~30 мин)

Обсуждаем с группой, какие задачи они обычно выполняют в GitLab. Берем реальные сценарии участников и пробуем выполнить их через OpenCode прямо на занятии.

## Часть 4. Кейс: синхронизация документации (~10 мин)

Рассказ о реальном применении: как OpenCode + GitLab используется для синхронизации старой версии документации с новой -- агент сравнивает структуру, находит расхождения, предлагает обновления.
