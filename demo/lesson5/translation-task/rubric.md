# Рубрика оценки качества перевода технической документации (RU -> EN)

Упрощённая рубрика для учебного задания. 4 критерия, шкала 0-5, итоговый балл -- взвешенная сумма.

## Принципы

- Оценка 5 -- ТОЛЬКО при полном отсутствии замечаний
- Каждая оценка ДОЛЖНА сопровождаться конкретными примерами (цитатами)
- Множественные мелкие ошибки снижают балл сильнее, чем одна

---

## 1. Соответствие глоссариям (вес: 0.35)

Термины переведены точно так, как указано в приложенных глоссариях (glossary-product.md и glossary-security.md). Любое отклонение от глоссария -- ошибка.

| Балл | Описание |
|------|----------|
| **5** | Все термины из глоссариев переведены точно как указано. Ноль отклонений. |
| **4** | 1-2 отклонения от глоссария, не искажающих смысл (допустимый синоним). |
| **3** | 3-5 отклонений от глоссария, ИЛИ 1-2 случая, когда термин переведён неверно. |
| **2** | Систематическое игнорирование глоссария: более 5 отклонений, ИЛИ ключевые термины переведены не по глоссарию. |
| **1** | Глоссарий практически не использован. Большинство терминов переведены произвольно. |
| **0** | Терминология не имеет отношения к глоссарию. |

## 2. Смысловая точность (вес: 0.30)

Полнота и точность передачи смысла. Ничего не потеряно, не добавлено, не искажено.

| Балл | Описание |
|------|----------|
| **5** | Смысл передан полностью и точно. Нет пропусков, искажений или добавлений. |
| **4** | 1-2 незначительных смысловых отклонения, не влияющих на понимание. |
| **3** | 1-2 предложения с изменённым смыслом, ИЛИ пропущен фрагмент, ИЛИ добавлена отсутствующая информация. |
| **2** | 3+ предложений с изменённым смыслом, ИЛИ пропущен абзац, ИЛИ искажены причинно-следственные связи. |
| **1** | Основной смысл утрачен. Многочисленные грубые искажения. |
| **0** | Перевод не соответствует исходному тексту. |

## 3. Естественность (вес: 0.20)

Текст читается как нормальная англоязычная техническая документация, без калек с русского.

| Балл | Описание |
|------|----------|
| **5** | Читается как написанный носителем. Без ощущения перевода. |
| **4** | 1-2 неловкие формулировки или кальки. |
| **3** | 3-5 калек или неестественных оборотов. Текст звучит как перевод. |
| **2** | Систематические кальки. Канцелярит ("It is necessary to carry out the procedure of..."). |
| **1** | Стиль полностью чужд техдокументации. Требует полной переработки. |
| **0** | Текст нечитаем. |

## 4. Полнота и форматирование (вес: 0.15)

Все элементы на месте (заголовки, списки, примечания, код, ссылки). Разметка сохранена.

| Балл | Описание |
|------|----------|
| **5** | Все элементы присутствуют, форматирование идентично оригиналу. |
| **4** | 1-2 мелких пропуска или дефекта форматирования. |
| **3** | Пропущены 2-3 элемента, ИЛИ заметно нарушена структура. |
| **2** | Пропущен целый раздел, ИЛИ форматирование существенно сломано. |
| **1** | Значительная часть контента отсутствует. |
| **0** | Перевод фрагментарен, форматирование уничтожено. |

---

## Итоговый балл

```
Score = (K1 * 0.35) + (K2 * 0.30) + (K3 * 0.20) + (K4 * 0.15)
```

Где: K1 -- Соответствие глоссариям, K2 -- Смысловая точность, K3 -- Естественность, K4 -- Полнота и форматирование.

| Диапазон | Качество |
|----------|----------|
| 4.5 - 5.0 | Отличное -- пригоден без редактуры |
| 3.5 - 4.4 | Хорошее -- минимальная вычитка |
| 2.5 - 3.4 | Удовлетворительное -- нужна существенная редактура |
| 1.5 - 2.4 | Низкое -- проще перевести заново |
| 0.0 - 1.4 | Непригодное |

## Инструкция для LLM-оценщика

```
Ты -- строгий эксперт по оценке качества перевода технической документации.

КОНТЕКСТ: Перевод выполнен с опорой на два глоссария терминов. Глоссарии приложены ниже.

ЗАДАЧА: Оцени качество перевода по 4 критериям (шкала 0-5).

ПРАВИЛА:
1. Оценка 5 -- ТОЛЬКО если нет НИ ОДНОГО замечания. Это редкая оценка.
2. Оценка 3 -- средний, приемлемый перевод.
3. Для КАЖДОГО критерия ОБЯЗАТЕЛЬНО приведи конкретные примеры (цитаты из оригинала и перевода).
4. По критерию "Соответствие глоссариям" -- сверяй КАЖДЫЙ специализированный термин с глоссариями. Любое отклонение -- ошибка.

ЗАПРЕЩЕНО:
- Ставить 5 по всем критериям
- Ставить оценки без цитирования примеров
- Игнорировать отклонения от глоссария
```
