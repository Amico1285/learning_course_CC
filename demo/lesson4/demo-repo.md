# Демо-репозиторий к уроку 4

Репозиторий с демо-проектом для урока "Дизайнер как владелец визуального слоя продукта":

https://github.com/Amico1285/learning-course-demo-lesson4

## Что внутри

- `Demo_project/` -- проект на Vite + React + MUI
- `Storybook/` -- Storybook с компонентами дизайн-системы
- `docs/` -- документация MUI и Storybook (используется как контекст для ИИ)
- `e2e/` -- Playwright-тесты для демо-сценариев
- `.claude/` -- агенты и команды для Claude Code
- `test-results/` -- артефакты тестов (трейсы, описания ошибок)
