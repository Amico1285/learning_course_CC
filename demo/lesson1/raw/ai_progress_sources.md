# Источники: Прогресс AI моделей

## Исследование проведено: 3 февраля 2025

## Основные источники

### 1. Epoch AI Trends Dashboard
- **Что:** Единый дашборд с метриками прогресса AI
- **Данные:** 
  - Training compute растет 100,000x в год с 2020
  - Algorithmic efficiency: снижение compute на 1000% в год
  - FLOP/s per dollar: улучшение на 23.4x в год
  - LLM context windows: рост в 1e30x с 2023
- **URL:** https://epoch.ai/trends

### 2. MMLU (Measuring Massive Multitask Language Understanding)
- **Что:** Стандартный бенчмарк для оценки языковых моделей
- **Данные:**
  - Random baseline: 25%
  - GPT-2: 32.4%
  - GPT-3 (few-shot): 43.9%
  - UnifiedQA: 48.9%
  - Современные модели: 90%+ (выше человеческого уровня)
- **URL:** https://arxiv.org/abs/2009.03300

### 3. HELM (Holistic Evaluation of Language Models)
- **Что:** Стандартизированная оценка от Stanford CRFM
- **Что оценивает:** 7 метрик (accuracy, calibration, robustness, fairness, bias, toxicity, efficiency)
- **Улучшение:** С 17.9% до 96.0% покрытия core scenarios
- **Paper:** arXiv:2211.09110

### 4. HumanEval
- **Что:** Бенчмарк для оценки генерации кода
- **Данные:** Современные модели достигают 90%+ на многих задачах
